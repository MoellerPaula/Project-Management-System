# Project Management System
A Spring Boot-based application for managing software projects, tasks and team collaboration.

## Overview
This project aims to develop a backend application for managing software projects and development workflows.

The system is inspired by existing project management tools and focuses on typical processes in software development teams, such as task management, project organization and progress tracking.

This project focuses on building a well-structured backend application while applying professional software development practices, including clean architecture, automated testing and modern development workflows.

## Planned Features
### Project Management
- Create and manage software projects
- Assign users to projects
- Track project progress

### Task Management
- Create, update and delete tasks
- Filter and search tasks
- Assign tasks to team members
- Manage task priorities
- Track task status

### User Management
- Manage users
- Define project roles
